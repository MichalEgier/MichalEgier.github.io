Motion Simulator

Technologie: C, Java, Bash
Technologies: C, Java, Bash

Motion simulator jest programem, który rozwiązuje zadane równania ruchu metodą numeryczną, łącznie z problemami mającymi zmienną siłe, zależną od którejś składowej prędkości, położenia bądź też czasu. Po pierwsze, pokazuje on na ekranie stan obiektu (prędkość, siła, położenie) po określonym w warunkach problemu czasie. Wszystko jest wpisywane z klawiatury po uruchomieniu programu, należy przestrzegać wpisywania w ustalonym formacie, jeżeli wpisywane są siły będące funkcją prędkości, położenia czy czasu. Program wizualizuje również animację trajektorii w dwóch wymiarach (x oraz y), lecz w tej wersji nie naprawia problemu z obiektem wychodzącym poza ekran, jeżeli położenie w pewnym momencie okaże się zbyt mało bądź zbyt duże. Skrypty w języku bash zostały użyte w programie, dlatego do uruchomienia go na systemie Windows potrzebne będzie zewnętrzne oprogramowanie emulujące powłokę bash.
Motion simulator is a program that solves motion problems, including problems that have force dependent on velocity or location. First it shows on console the state of object (velocity,force,location) after time specified by an user. Everything is written on a keyboard ath the beggining of the program, an user must care about writting in specified format and avoid unspecified characters. It also shows visualisation of trajectory in 2 dimensions (x and y), but doesn't fix problems with object going out of window, if location is too big or too small. Note that Bash scripts are used in the project so on Windows operating system it will need some external programs to run.




Po lewej przedstawiony jest na animacji proces wpisywania danych przez użytkownika, zrealizowany w skrypcie Bashowym, który następnie przekazuje argumenty jako parametry wywołania do silnika napisanego w języku C. Jako input z założenia podawana jest tu przez użytkownika funkcja siły oraz warunki początkowe, tj. początkowa prędkość oraz początkowe położenie ciała, będące konieczne do wyliczenia stanu końcowego. Po zakończeniu etapu obliczeń, skrypt przechodzi do etapu prezentacji, odpalając program napisany w języku Java, służący do wizualizacji wyników. Czyta on dane z odpowiedniego pliku, będącego wynikiem pracy wcześniejszego silnika w języku C.
On the left there is an animation, that presents a process of specifying data by an user, realised in Bash script, that afterwards pass arguments to engine (written in C language) as calling parameters. As input, from assumption, an user specifies: a function of force and initial conditions, so initial velocity and initial position of body. Those values are neccesery to compute the final state. After completing the computing part, script goes to the presentation part. It runs the program written in Java language, which is supossed to visualise previously computed results. This Java program reads data from a proper file, that is prepared before by C-language engine.






Po prawej widać rozwiązanie dla problemu klasycznego oscylatora harmonicznego, bez wymuszania drgania oraz bez jego tłumienia. Przedstawionymi na ekranie wektorami są zarówno wektor prędkości, jak i wektor siły działającej w danym momencie na ciało.
On the right we see the solution for a problem of classic harmonic oscillator, without forcing of oscillation or vibration damping. Visibile on the screen vectors are a vector of velocity of a body as well as a vector of force that acted on a body in the current moment of time.









Po lewej widać rozwiązanie dla problemu rzutu ukośnego z wliczeniem oporu atmosferycznego zależnego od prędkości ciała.
On the left we see the solution for a problem of oblique throw, including speed dependent atmospheric resistance.

Sensor Paint

Technologie: platforma android, Java
Technologies: platforma android, Java

Sensor paint jest aplikacją przypominającą klasycznego painta, ale w której nawigacja (poruszanie kursorem) zrealizowana jest za pomocą sensorów sprzętowych zainstalowanych w większości smartphonów. Konkretniej są to sensory rotacyjne (Rotation sensors), wykrywające obracanie telefonu wzdłuż osi współrzędnych. Komunikacja z sensorami w głównej mierze została zaimplementowana przy wykorzystaniu wzorca projektowego obserwator, odpowiednio reagującego na kolejne informacje z interfejsu sensorów. Naciśnięcie ekranu spowoduje rysowanie na ekranie w miejscu aktualnego położenia kursora.
Sensor paint is the application similar to the classic Paint, but navigation (moving the cursor) is realised with hardware sensors installed in most of smartphones. More specifically, it is a rotation sensor, that detects the angular motion (rotating) around the axes. Communication with sensors was mainly implemented using Observer pattern, that provides appriopriate reacting for consecutive informations from sensors interface. Pressing the screen will result in drawing on the screen in the current cursor position.









Aplikacja posiada również wbudowany system zapisywania oraz wczytywania stworzonych obrazków z pamięci telefonu, oraz dostosowania wielkości kursora oraz jego koloru, a także zmianę koloru tła. Zastosowane są trzy podręczne przestrzenie robocze (z wykorzystaniem fragmentów oraz zakładek), zachowujące wszystkie ustawienia lokalnie.
Application also has a system of loading and saving in phone memory pictures made by an user, it allows to customize the size of a cursor and its color, and also to customize a color of the background. There are three workspaces (realised using fragments and tabs), that preserve all local settings.



GA Circuit Board

Technologie: Java
Technologies: Java

GA (Genetical algorithm) Circuit Board to program służacy do wyszukiwania (jest to heurystyka, a więc bez gwarancji sukcesu) takich połączeń na zadanej płytce, aby się nie przecinały. Jako wejście program przyjmuje odpowiednio sformatowane dane w postaci wielkości płytki, a także zbioru par punktów A-B, które mają zostać połączone. Zrealizowane zostało to przy pomocy algorytmu genetycznego (sztuczna inteligencja). Na rysunku po prawej znaleziono akurat z przecięciem w jednym miejscu - wynika to z tego, że uruchomiono algorytm wyjątkowo krótki czas w celu prezentacji.
GA (Genetical algorithm) Circuit Board is a program intended to search (it is heuristic, so without any guarantee of success) for a set of connections on a specified board, that causes no crosscuts between any two of connections. As input program takes appriopriately formatted data representing size of board, and also set of pairs A-B, that are supossed to be connected. Program is realised with genetical algorithm approach (artifical inteligence). In the picture on the right it actually found the solution with one crosscut - it is due to the fact, that program has been run for a very small amount of time only for the presentation purposes, and genetical algorithm from assumption gives the better solution the more time it works.





W projekcie zastosowano wzorce projektowe takie jak: strategia, która posłużyła wydzielenie odpowiedzialności takich jak mutacja, selekcja oraz krzyżowanie, co pozwoliłoby także na dalszą rozbudowę programu, a także łatwą wymianę każdej z tych metodyk na inna, oraz MVC, dzięki czemu wydzielono warstwę modelu, widoku (graficzną) oraz kontrolera.
In the project there were used project patterns such as: Strategy, that was used to divide and isolate responsibilities such as mutation, selection and crossovering, what would let us further develop the program, and also easily replace one of the methods with another. Model, view and controller layers were specified using MVC pattern.



Mini eShop application

Technologie: platforma .Net, C#, HTML
Technologies: .Net platform, C#, HTML

Aplikacja ta jest bardzo uproszczoną imitacją sklepu internetowego dostępnego przez przeglądarkę. Zaimplementowany został tutaj system kont, realizujący również kontrolę dostępu, jaki przydzielono odpowiednim użytkownikom (Identity framework), a także przechowywanie produktów sklepu dodanych przez administratora w bazie danych (Entity framework).
This application is a very simplified imitation of an eShop accesed by a webbrowser. There has been a system of account implemented, that realises also access control (Identity framework), and also storing shop products added by administrator in database (Entity framework).







Po lewej próba wpisania zbyt słabego hasła oraz próba wykonania czynności wymagającej uprawnień administratora (widoczna tutaj zakładka Admin jest oczywiście tutaj tylko po to aby pokazać faktyczne blokowanie dostępu; w komercyjnej aplikacji wyglądałoby to oczywiście inaczej).
On the left we see an attempt to set password that is verified as too weak, and also attempt to perform activity that requires administrator privileges (visible Admin tab here is obviously only for presentation, showing that blocking of access was indeed performed; in comercial application it would of course look different).









Dodawanie produktów do bazy danych przez administratora.
Adding products to database by an administrator.







Dodawanie przedmiotów do koszyka, przeglądanie przedmiotów po kategoriach, a także próba zfinalizowania płatności, która wymagała zalogowania się jako zarejestrowany użytkownik (bądź zarejestrowania).
Adding products to the cart, searching products by categories, and also attempt to finalize payment, what requires loging in as a registered user (or creating a new account).
